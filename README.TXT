INSTALL SUPPORT LIBRARIES
    The game was developed with Simple Directmedia Layer 2.
    In order to run the program, please install libsdl2-2.0-0, libsdl2-image-2.0-0, libsdl2-mixer-2.0-0, libsdl2-ttf-2.0-0
    For convenience just execute sh run_debug.sh
